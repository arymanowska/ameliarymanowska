	installation - manual

	instalacja:
1) (folder powinien być zrobiony na dysku C) wpisujemy npx create-next-app@latest w terminalu
2) dostosowujemy opcje do naszych potrzeb i czekamy do końca instalacji
3) uruchamiamy aplikacje za pomocą npm run dev
4) npm install tailwindcss-animate class-variance-authority clsx tailwind-merge
5) pobieramy paczke z ikonami npm install lucide-react
6) dorzucamy "baseUrl": ".", w jsconfig.json
7) usuwamy kod z pliku tailwind.config.js i wklejamy ten z z instalacji manualnej (punkt 5 Configure tailwind.config.js)
8) usuwamy zawartość global.css i wklejamy kod z punktu 6
9) tworzymy w src folder "lib" a w nim plik utils.js
10) w pliku utils.js wklejamy kod z punktu 7 i usuwamy ClassValue
11) w tailwind.config.js zmieniamy content z ts i tsx na js i jsx i "app/**/*.{js,jsx}" na "./src/**/*.{js,jsx}"
12) tworzymy folder components w src

	getting started - components.json
1) npx shadcn@latest init

	pobieranie buttona:
1) npx shadcn@latest add button w terminalu
2) <Button variant="outline">Button</Button> w page.js 
3) import buttona

	button z ikoną:

1) export function ButtonIcon() {
  return (
    <Button variant="outline" size="icon">
      <ChevronRight className="h-4 w-4" />
    </Button>
  )
}
2) import { Button } from "@/components/ui/button"
3) import { ChevronRight } from "lucide-react"
